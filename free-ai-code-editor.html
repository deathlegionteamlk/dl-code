What makes DL Code a real AI code editor?

Most "AI code editors" are just regular editors with a chat sidebar bolted on. DL Code is different — AI is built into the editor itself. You get inline code suggestions (Copilot-style), a chat panel with full file context, an agentic app builder that creates entire projects from a prompt, and the ability to run Claude Code as an autonomous agent that can read/write files and run commands in your workspace.

Three ways to use AI in DL Code

1. Free unlimited AI via Ollama (recommended)

Run AI models locally on your machine with Ollama. No API key, no rate limits, no usage caps — completely free forever. Works offline. Recommended models:

  • Qwen 2.5 Coder (32B) — best overall for coding
  • DeepSeek Coder v2 — strong alternative

2. Claude Code Agent (most powerful)

Install the Claude Code CLI and DL Code runs it as an autonomous agent that can:

  • Read and write files in your workspace
  • Run shell commands
  • Edit code in place
  • Multi-step reasoning for complex refactors
npm install -g @anthropic/claude-code

3. Anthropic API (direct)

For quick inline suggestions without installing anything locally. Add your API key in Settings → AI. Supports Claude Sonnet 4.5, Opus 4.1, and Haiku 4.5.
