Features

Everything you need to ship code faster

A modern code editor inspired by JetBrains IDEs and the extensibility of VS Code — but with AI built in from the ground up.

Syntax Highlighting for 60+ Languages

TypeScript, JavaScript, Python, Rust, Go, Java, C++, C#, Ruby, PHP, Swift, Kotlin, Scala, HTML, CSS, JSON, YAML, SQL, Docker, Lua, Haskell, Elixir, Erlang, Clojure, Perl, Dart, R, Julia, OCaml, and many more — all with proper syntax highlighting powered by the Monaco editor.

Built-in AI Assistant

Chat with Claude right inside the editor. Ask it to explain code, refactor functions, generate tests, or fix bugs. The AI sees your file context, your selection, and your workspace — so it gives you answers that actually fit your codebase.

Agentic App Builder

Describe an app in plain English — "build me a todo list with React and TypeScript" — and DL Code's AI agent scaffolds the project, writes every file, and installs dependencies. A real Cursor-style builder that runs on your machine.

Real Integrated Terminal

A real PTY-backed terminal running bash, zsh, PowerShell, or cmd.exe — not a fake console. Multiple tabs, full color support, web links, and search. Run your builds, git commands, and tests without ever leaving the editor.

Git Integration

Source control panel with staged and unstaged changes, one-click stage/unstage, commit, branch switching, diff viewer, and full history. DL Code uses the real simple-git library under the hood — no shelling out to git CLI.

Plugin & Extension System

A real extension host that dynamically loads extensions from disk. Build your own themes, language packs, formatters, linters, and tools using a VS Code-compatible API. Ship extensions as folders — no packaging required.

Cross-File Search

Find anything across your entire workspace with regex support, case sensitivity, whole word matching, and file include/exclude globs. Results show context lines around each match — click to jump straight to the file.

Command Palette & Quick Open

Press Ctrl+Shift+P for the command palette. Press Ctrl+P to jump to any file. DL Code respects your muscle memory from VS Code and JetBrains — every shortcut just works.

Cross-Platform

One editor, three platforms. DL Code runs natively on Windows 10+, macOS (Intel and Apple Silicon via Rosetta), and Linux (AppImage, .deb, and portable tarball). Your settings, themes, and extensions sync across machines.

FAQ

Questions, answered

Is DL Code really free?

Yes — DL Code is 100% free, forever. There are no subscriptions, no usage caps, and no ads. The AI features work with local models via Ollama, which is also free and unlimited. If you want to use Claude (Anthropic's API), that's a separate paid service from Anthropic — but you don't need it to use DL Code.

How is DL Code different from VS Code?

DL Code uses the same Monaco editor as VS Code, so the editing experience feels familiar. The differences: (1) AI is built in from the start — no extension needed. (2) The agentic builder can scaffold entire projects from a prompt. (3) The UI is inspired by JetBrains IDEs, with a darker, denser, more focused layout. (4) Free unlimited AI via Ollama works out of the box.

Does DL Code support my programming language?

Probably — DL Code ships with syntax highlighting for 60+ languages including TypeScript, JavaScript, Python, Rust, Go, Java, C, C++, C#, Ruby, PHP, Swift, Kotlin, Scala, HTML, CSS, JSON, YAML, SQL, Docker, Lua, Haskell, Elixir, Erlang, Clojure, Perl, Dart, R, Julia, OCaml, and many more. If your language isn't supported, you can add it via an extension.

How does the agentic builder work?

You describe an app in plain English ("build me a todo list with React and TypeScript"). DL Code's AI agent — powered by Claude Code CLI, Ollama, or the Anthropic API — generates the project structure, writes all the code files, creates a README, and runs the install command. The whole thing happens on your machine; nothing is sent to a cloud server (unless you pick the Anthropic API).

Can I use DL Code offline?

Yes — if you pick the Ollama provider for AI, DL Code works completely offline. Ollama runs AI models locally on your machine, so you can code on a plane, on a train, or anywhere with no internet. The editor itself, the terminal, Git, and search all work offline too.

Is my code sent to the cloud?

Only if you explicitly choose the Anthropic API provider. With Ollama, everything stays on your machine. With Claude Code CLI, the agent runs locally but may make API calls to Anthropic as part of its reasoning — same as running Claude Code in your terminal.

Can I install extensions?

Yes — DL Code has a real extension host. Extensions live in userData/extensions/<publisher.name>/ as folders with a package.json and main.js. The API is compatible with VS Code's extension API, so many VS Code extensions can be ported with minimal changes.

What's the system requirements?

Any 64-bit Windows 10+, macOS 10.15+, or Linux distro from the last 5 years. 4GB RAM minimum, 8GB recommended. For local AI via Ollama, you'll want 8GB+ RAM (16GB+ for larger models).
