features

Everything the editor should've shipped with

Monaco underneath, JetBrains-style chrome on top, and AI wired in from day one instead of duct-taped on later.

60+ languages, properly highlighted

TypeScript, Python, Rust, Go, Java, C++, C#, Swift, Kotlin, SQL, Docker, Haskell, Elixir, and a lot more — all through Monaco, the same engine VS Code runs on.

An AI assistant that's actually in the editor

Chat with Claude without leaving your file. Ask it to explain a function, refactor, write tests, or find a bug — it sees your open file, your selection, your workspace.

An agentic app builder

Describe an app in plain English and DL Code scaffolds the project, writes every file, and installs the dependencies. Runs on your machine, not someone else's server.

A terminal that's actually a terminal

PTY-backed, running bash, zsh, PowerShell, or cmd.exe. Tabs, full color, clickable links, search. Not a text box pretending to be a shell.

Git, without switching windows

Staged and unstaged changes, one-click stage, commit, branch switching, diff viewer, full history — run on the real simple-git library, not a shell-out hack.

A real extension system

An extension host that loads from disk. Build themes, language packs, linters, formatters with a VS Code-compatible API. Ship as a folder — no packaging step.

Search across the whole workspace

Regex, case sensitivity, whole word, file globs. Context lines around every match — click one and you're in the file.

Command Palette & Quick Open

Ctrl+Shift+P for commands. Ctrl+P to jump to any file. Your VS Code and JetBrains muscle memory works here unmodified.

Windows, macOS, Linux

One codebase, three platforms: Windows 10+, macOS (Intel and Apple Silicon via Rosetta), and Linux as AppImage, .deb, or a plain tarball.

faq

Questions people actually ask

Is it really free?

Yes — 100% free, forever. No subscription, no usage cap, no ads. AI works with local models via Ollama, which is also free and unlimited. If you'd rather use Anthropic's API, that's a separate paid service from Anthropic — but you don't need it to use DL Code.

How is this different from VS Code?

Same Monaco editor underneath, so it'll feel familiar. The differences: AI is built in, not an extension you install. The agentic builder scaffolds whole projects from a prompt. The UI leans JetBrains — darker, denser. And Ollama gives you free unlimited AI out of the box.

Does it support my language?

Almost certainly — 60+ languages ship with syntax highlighting out of the box, including TypeScript, Python, Rust, Go, Java, C++, Swift, Kotlin, SQL, and Docker. Missing one? Add it through an extension.

How does the agentic builder actually work?

You describe an app in plain English. The AI agent — Claude Code CLI, Ollama, or the Anthropic API, your pick — writes the project structure, every code file, a README, and runs the install command. It all happens on your machine; nothing leaves it unless you've chosen the Anthropic API.

Can I use it offline?

Yes, with Ollama selected. Models run locally, so you can code on a plane with no signal. The editor, terminal, Git, and search all work offline regardless of which AI provider you pick.

Is my code sent anywhere?

Only if you choose the Anthropic API provider. With Ollama, everything stays on your machine. With Claude Code CLI, the agent runs locally but does call Anthropic's API as part of its reasoning — same as running Claude Code in a terminal.

Can I install extensions?

Yes. Extensions live in userData/extensions/<publisher.name>/ as folders with a package.json and main.js. The API matches VS Code's closely enough that a lot of existing extensions port with minimal changes.

What do I need to run it?

64-bit Windows 10+, macOS 10.15+, or a Linux distro from the last few years. 4 GB RAM minimum, 8 GB is more comfortable. Running Ollama locally? Budget 8 GB+, 16 GB+ for the bigger models.
