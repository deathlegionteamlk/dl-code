How to install DL Code on macOS

  1. Download DL-Code-1.0.0-macos-x64.zip from the Releases page
  2. Unzip the file — you'll get DL Code.app
  3. Drag DL Code.app to your Applications folder
  4. Right-click the app and select Open (first time only, to bypass Gatekeeper)
  5. Click Open again in the warning dialog

After the first launch, you can open DL Code normally from Launchpad or Spotlight.
