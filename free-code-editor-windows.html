How to install DL Code on Windows

  1. Download DL-Code-1.0.0-0.0.1-win-x64-portable.zip from the Releases page
  2. Unzip the file to any folder (e.g. C:\Tools\DL-Code)
  3. Double-click DL Code.exe to launch
  4. Optional: Pin to taskbar or create a desktop shortcut

No installer, no admin rights required, no registry changes. DL Code is fully portable.
